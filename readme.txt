This plugin once activated enables a popup of related posts. The popup appears when user is about to change the site URL or close current tab.
 
== Description ==
 
'Before exit popup' provides backend options to select whether to show related posts or the selected posts.
To show selected posts you will have to enter comma seperated string of post IDs. These posts will be shown in the popup.

Actually there is no way to find out when user will leave this site.
Before Exit Popup tries to get user attention back to site by showing a popup when the cursor is moved to browser control tab. 
This plugin does not stop user from closing the browser or tab. 
 
 
== Installation ==
 
This section describes how to install the plugin and get it working.

1. Upload before-exit-popup to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
